## ADDED Requirements

### Requirement: Generated appendix labeled for Chinese users
The VitePress API Reference navigation entry for the TypeDoc appendix SHALL make clear that the appendix is auto-generated from source comments and is intended as a Chinese-annotated symbol reference (wording may vary), not an English-only dump.

#### Scenario: Nav or landing explains appendix
- **WHEN** a reader opens the generated appendix landing or its nav label context
- **THEN** they see a short note that content comes from source JSDoc and is regenerated by `docs:api`
