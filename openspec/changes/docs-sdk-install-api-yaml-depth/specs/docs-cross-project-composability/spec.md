## ADDED Requirements

### Requirement: Multilang install linked from cross-project guide
The cross-project guide SHALL link to (or briefly summarize) language-specific install commands and a TS-parent / other-language-child Unit example so readers do not need to discover packaging only via cross-lang demos.

#### Scenario: Cross-project page mentions install
- **WHEN** a reader opens the cross-project guide to embed another project
- **THEN** they are directed to install steps for the parent SDK and how the child exposes `/execute`
